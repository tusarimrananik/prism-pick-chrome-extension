## Detailed description

Prism Pick is a fast, focused color picker for designers, developers, students, and creators.

Pick any visible color from a webpage with Chrome's native eyedropper, then copy the result as HEX or RGB in one click. Prism Pick also keeps your five most recent colors available for quick reuse.

Features:

- Pick colors directly from webpages
- Copy HEX values in one click
- Copy RGB values in one click
- Reuse five recent colors
- Clear color history whenever you want
- Accessible keyboard controls and visible focus states
- Clean, compact interface

Privacy comes first. Prism Pick has no analytics, advertising, accounts, or tracking. Recent colors are stored only in your browser using Chrome's local extension storage.

Prism Pick requests only the `storage` permission, which is used to remember recent colors on your device.

## Single purpose

Prism Pick lets users sample visible colors and copy their HEX or RGB values.

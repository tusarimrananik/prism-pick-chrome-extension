# Chrome Web Store Privacy Answers

Use these answers in the Privacy practices tab and verify them again before submission.

## Single purpose description

Prism Pick lets users sample visible colors and copy their HEX or RGB values.

## Permission justification

### storage

Used to save up to five recent color values locally in the user's Chrome profile so they remain available the next time the popup opens.

## Data usage certification

- Personally identifiable information: Not collected
- Health information: Not collected
- Financial and payment information: Not collected
- Authentication information: Not collected
- Personal communications: Not collected
- Location: Not collected
- Web history: Not collected
- User activity: Not collected
- Website content: Not collected or transmitted

The extension does not sell or transfer user data, use data for purposes unrelated to its single purpose, or use data for creditworthiness or lending.

## Remote code

No remote code is used. All executable code is packaged with the extension.

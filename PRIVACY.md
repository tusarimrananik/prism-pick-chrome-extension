# Privacy Policy for Prism Pick

**Effective date:** July 18, 2026

Prism Pick is designed to work without collecting personal information.

## Data stored on your device

Prism Pick stores up to five recently selected color values using Chrome's local extension storage. This information stays in the user's browser profile and can be cleared from the extension's settings menu or by uninstalling the extension.

## Data collection and sharing

Prism Pick does not collect, transmit, sell, rent, or share personal information. It does not use analytics, advertising, tracking technologies, remote code, or third-party services.

## Permissions

The extension requests only the `storage` permission. It is used exclusively to remember recent color values locally. Color sampling is performed through Chrome's native EyeDropper API after the user presses the pick button.

## Changes

If this policy changes, the updated version will be published in this repository with a new effective date.

## Contact

For privacy questions, open a GitHub issue in the Prism Pick repository.
